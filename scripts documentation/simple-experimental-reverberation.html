What this does

Praat script.

Process Description: The Smooth Cosmic Reverb Effect script generates a lush, highly diffused reverberation that focuses on smooth, slow modulation and rich complexity. Instead of relying on raw randomness (jitter), it uses controlled, sinusoidal modulations in the delay times and amplitudes to create a feeling of shimmering, evolving space—the "cosmic" element.

Quick start

  1. In Praat, select a Sound object.
  2. Run script… → Simple_Experimental_Reverberation.praat.
  3. Set parameters and click OK.
  4. Output object appears (see Outputs).
