What this does

Praat script.

Process Description: The 8-Channel Delay Processor script takes a mono sound source and applies eight unique delay-differential filters with extremely short delay times, treating each delayed signal as a separate channel. It then combines all eight channels into a final StereoOut object. The primary function of this script is to generate a dense, phase-based, and metallic texture by exploiting the psychoacoustic effects of micro-delays and phase inversion.

Technical details: This Praat script implements an 8-channel comb filter delay processor with alternating polarity architecture that generates complex spectral interference patterns and stereo spatial effects through sample-level delay manipulation and selective time reversal. The algorithm operates on a mono source signal, creating eight independent processing channels where each applies a user-defined sample delay (default values ranging from 2 to 24 samples). For each channel, the delay length is specified directly in samples, and the core transformation applies a differential comb filtering equation: output[n] = input[n + delay] - input[n], which creates frequency-dependent notches by subtracting the current sample from its time-shifted counterpart. At the buffer boundary where n + delay exceeds the signal length, the formula inverts to -input[n], ensuring continuous processing without array overflow while introducing phase reversal at the signal tail. The script then applies selective time-domain reversal to even-numbered channels (2, 4, 6, 8) using Praat's reverse operator, which performs complete temporal inversion by reordering samples from end to beginning. This alternating forward-reverse architecture creates asymmetric stereo imaging when the eight channels are downmixed, as reversed channels exhibit inverted transient envelopes and phase relationships relative to forward channels. The differential delay processing emphasizes high-frequency content and transients while creating harmonic resonances at frequencies inversely proportional to the delay times (approximately sample_rate / delay Hz), and the channel-specific delays produce distinct comb filter notch patterns that combine into a rich, metallic timbre. All eight processed channels are combined into a single stereo object through Praat's multi-object stereo combination, which performs implicit summation and channel interleaving, followed by peak normalization to a user-specified level (default 0.99) to prevent clipping. The alternating reversal pattern creates pseudo-spatial effects through decorrelation between odd and even channel groups, producing a wide stereo image with complex phase relationships and time-varying interference patterns characteristic of flanging and phasing effects.

Quick start

  1. In Praat, select a Sound object.
  2. Run script… → 8 - Odd forward Even reversed.praat.
  3. Set parameters and click OK.
  4. Output object appears (see Outputs).
