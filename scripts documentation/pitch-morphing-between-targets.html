What this does

Praat script.

Process Description: This Praat script executes an advanced Pitch Morphing effect by sequentially transitioning the sound's fundamental frequency (F0) through a series of user-defined semitone targets. It employs sophisticated mathematical curves—including elastic smoothing, dynamic overshoot, and tension modulation—to create highly dynamic and non-linear pitch transitions, applied via Time-Domain Pitch Synchronous Overlap and Add (TD-PSOLA) resynthesis.

Quick start

  1. In Praat, select a Sound object.
  2. Run script… → PITCH_MORPHING_BETWEEN_TARGETS.praat.
  3. Set parameters and click OK.
  4. Output object appears (see Outputs).
