What this does

Praat script.

Process Description: Advanced Brownian Synthesis System designed for sound design and generative composition, creating textures or drones where the core frequency dynamically shifts based on random walk (Brownian motion). It then offers sophisticated multi−layer synthesis and four different spatial processing modes.

Technical details: This Praat script implements a multi-mode stochastic synthesis engine that generates evolving soundscapes through Brownian motion-controlled frequency trajectories across multiple independent oscillator layers. The algorithm operates at dual time scales: audio-rate synthesis at 44.1 kHz sample rate and control-rate parameter updates at 40 Hz (25 ms intervals), with each layer maintaining phase continuity through cumulative phase tracking (phase_n+1 = phase_n + 2π × frequency × Δt). The core Brownian walk mechanism updates oscillator frequencies via additive random steps drawn from Gaussian distributions (freq_n+1 = freq_n + N(0, step_size)), with optional drift correction that applies restorative force proportional to deviation from target frequencies (drift = (target - current) × drift_force × Δt), preventing extreme frequency excursions while maintaining stochastic character. Five synthesis modes implement distinct algorithmic variations: Basic Walk uses constant-amplitude sinusoids with frequency wandering; Chaos Mode introduces occasional large jumps (5× normal step with 10% probability) and stability-based amplitude modulation (amp = exp(-|step|/chaos_factor)); Filtered Brownian applies time-varying bandpass filters to white noise with center frequencies following Brownian trajectories; Harmonic Mode generates harmonic series (integer frequency ratios) with independent Brownian deviations per partial and 1/n amplitude falloff; and Pulsed Mode gates amplitude through rhythmic envelopes (pulse = (sin(2π × pulse_rate × t) > 0.7) × 0.8 + 0.2). Each layer's waveform is constructed through piecewise formula concatenation, building conditional expressions that define time-windowed sinusoidal segments with continuously varying frequencies, then combined through normalized summation (output = Σ(layer_i) / N). Four spatial modes provide stereo imaging: Wide creates static stereo through complementary spectral filtering (left: DC-4kHz, right: 200Hz-8kHz); Rotating implements circular panning via orthogonal amplitude modulation (left = 0.6 + 0.4×cos(2πft), right = 0.6 + 0.4×sin(2πft)); Binaural simulates head-related transfer functions through asymmetric filtering and 30-sample interaural time delay; Mono preserves single-channel output. The system applies linear fade envelopes at boundaries using sample-domain multiplication (amp = min(t, duration-t) / fade_time), frequency clamping prevents oscillator runaway (typical range 20-8000 Hz depending on mode), and optional peak normalization to 0.9 prevents clipping. This architecture produces organic, continuously evolving textures characteristic of natural acoustic phenomena—from subtle pitch drift resembling analog oscillator instability to complex inharmonic clouds suitable for ambient composition, drone music, and generative sound design where controlled randomness creates perpetually non-repeating sonic material.

Quick start

  1. In Praat, select a Sound object.
  2. Run script… → Advanced Brownian Synthesis.praat.
  3. Set parameters and click OK.
  4. Output object appears (see Outputs).
