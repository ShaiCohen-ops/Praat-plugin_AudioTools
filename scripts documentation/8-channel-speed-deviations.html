What this does

Praat script.

Process Description: The Create 8-channel speed-deviated file script generates a massive, time-stretched, and dense spatial effect by creating eight copies of the source audio, each playing at a slightly different, fixed speed. When combined, this creates a choir-like, asynchronous wash that can range from subtle detuning to complete temporal and harmonic chaos.

Quick start

  1. In Praat, select a Sound object.
  2. Run script… → 8-channel speed deviations.praat.
  3. Set parameters and click OK.
  4. Output object appears (see Outputs).
