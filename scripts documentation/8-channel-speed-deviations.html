What this does

Praat script.

Process Description: The Create 8-channel speed-deviated file script generates a massive, time-stretched, and dense spatial effect by creating eight copies of the source audio, each playing at a slightly different, fixed speed. When combined, this creates a choir-like, asynchronous wash that can range from subtle detuning to complete temporal and harmonic chaos.

Technical details: This Praat script implements an 8-channel temporal displacement processor that generates polyphonic texture through independent time-stretching of parallel audio streams. The algorithm operates in three modes: automatic generation using a deviation factor that linearly interpolates speed multipliers across channels (e.g., a factor of 0.15 produces speeds ranging from 0.85× to 1.15× across the eight channels); manual specification allowing precise control over each channel's playback rate; and pseudo-random generation using a deterministic seed-based algorithm (formula: (seed × channel × 137 + 97) mod 1000 / 1000) that produces reproducible yet varied speed distributions within user-defined bounds. The core time-stretching engine employs Praat's overlap-add (PSOLA) method with pitch range constraints (75-600 Hz) to modify duration without altering pitch, calculating target durations as original_duration / speed_factor where factors below 1.0 compress time and factors above 1.0 expand it. Each channel undergoes a three-stage transformation pipeline: first, a working copy is created and converted to mono if necessary; second, the overlap-add algorithm performs pitch-synchronous time modification by identifying periodic segments and redistributing them along the time axis to achieve the target duration while maintaining spectral integrity; third, all channels are resampled to a common sampling frequency (default 44.1 kHz) with 50-point sinc interpolation to ensure temporal alignment despite their different durations. The script includes an optional sampling frequency override that manipulates the source audio's declared sample rate before processing, effectively creating additional pitch shifting when combined with the time-stretching operations. Finally, the eight independently time-warped channels are combined into a single multi-channel sound object through iterative stereo combination, producing a dense rhythmic and temporal texture where identical source material plays at eight simultaneous speeds. This creates natural chorus, phasing, and polyrhythmic effects, with the speed distribution pattern (linear, manual, or random) determining the coherence and complexity of the resulting temporal relationships.

Quick start

  1. In Praat, select a Sound object.
  2. Run script… → 8-channel speed deviations.praat.
  3. Set parameters and click OK.
  4. Output object appears (see Outputs).
