What this does

Praat script.

Process Description: This script applies a multi-stage acoustic transformation to the selected sound, simulating the effect of passing the input audio through four different, fixed vocal tract models corresponding to the fundamental vowels /a/, /e/, /i/, and /o/. The four resulting filtered sounds are then joined sequentially. This process essentially treats the original sound as the excitation source and the vowel models as spectral filters, imparting the characteristic formant structure of each vowel onto the input audio's timbre.

Technical details: This Praat script implements a multi-vowel formant filtering system that processes input audio through four distinct vocal tract transfer functions, generating a concatenated sequence that traverses cardinal vowel spaces. The algorithm begins by instantiating four VocalTract objects representing the articulatory configurations for /a/, /e/, /i/, and /o/ phonemes, which encode area functions defining vocal tract geometry from glottis to lips with frequency-dependent resonance characteristics. These vocal tract models are bundled into a VocalTractTier spanning the input signal's duration (divided into 0.5-second interpolation segments) and converted to Linear Predictive Coding (LPC) objects through all-pole filter modeling with 5 ms analysis windows. The LPC representation approximates the vocal tract transfer function as an autoregressive model H(z) = G / (1 - Σ(a_k × z^(-k))) where coefficients a_k encode formant frequencies and bandwidths, typically using 10-16 poles for adult speech (automatically determined by Praat based on sample rate and expected formant count). Each LPC filter is then applied to the source audio through inverse filtering followed by synthesis filtering: the source signal is decorrelated through the inverse filter to approximate the glottal source, then reconstructed through the LPC synthesis filter representing the target vowel's spectral envelope. This process performs spectral reshaping where the source's harmonic or noise content is redistributed according to the vowel's formant structure—for example, /i/ emphasizes high first formant (F1 ~300 Hz) and very high second formant (F2 ~2300 Hz), while /a/ exhibits high F1 (~700 Hz) and mid-range F2 (~1200 Hz). The filtering operation Filter: "no" applies LPC synthesis without pre-emphasis, preserving the natural spectral balance of the vocal tract model. Following filtering, each vowel-processed sound undergoes resampling to the original sample rate with precision factor 50 (ensuring high-quality sinc interpolation to correct any sample rate discrepancies introduced during LPC conversion), then all four vowel-filtered versions are concatenated sequentially through Praat's temporal concatenation operator, which joins waveforms end-to-end without crossfading or overlap. The resulting output presents the source material filtered through four successive vowel colorations, creating a morphing effect where the same acoustic content exhibits /a/-/e/-/i/-/o/ timbral progression—particularly effective on harmonic sources (pitched instruments, vocals) where formant structure becomes prominent, but also applicable to noise sources where broadband energy is sculpted by vowel resonances. The script performs comprehensive cleanup by removing all intermediate objects (VocalTract, VocalTractTier, LPC, and pre-resampling Sound objects), retaining only the original input and final concatenated output. This technique finds applications in vowel synthesis, speech resynthesis from arbitrary excitation sources, formant-based timbral transformation, pedagogical demonstrations of source-filter theory, and creative sound design where vocal-like qualities are imposed on non-speech material, effectively making any sound 'sing' through the specified vowel sequence.

Quick start

  1. In Praat, select a Sound object.
  2. Run script… → a-e-i-o_filter.praat.
  3. Set parameters and click OK.
  4. Output object appears (see Outputs).
