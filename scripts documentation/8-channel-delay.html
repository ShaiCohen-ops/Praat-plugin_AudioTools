What this does

Praat script.

Process Description: The 8-Channel Delay Processor script generates a dense, massive, and highly unstable resonance by creating eight unique subtractive comb filters and combining them into a stereo field. Unlike simple echo effects, this script uses large, precisely calculated delays based on the total sound duration, resulting in a persistent, resonant wash of metallic, filtered sound.

Technical details: This Praat script implements an 8-channel comb filter delay processor that generates complex phasing and resonance effects through sample-level delay manipulation and differential subtraction. The algorithm operates on a mono source signal, creating eight independent processing channels where each applies a mathematically derived delay based on user-defined divisors. For each channel, the delay length is calculated as floor(total_samples / divisor), producing inversely proportional delay times—higher divisors yield shorter delays, enabling precise control over the temporal cascade. The core transformation applies a difference equation formula: output[n] = input[n + delay] - input[n], which creates a comb filtering effect by subtracting the current sample from its time-shifted counterpart. At the buffer boundary where n + delay exceeds the signal length, the formula inverts to -input[n], ensuring continuous processing without array overflow while introducing a phase reversal at the tail. This differential approach emphasizes transient information and creates frequency-dependent notches characteristic of feedforward comb filters. The eight processed channels—each with distinct delay characteristics ranging from aggressive (divisor 2) to subtle (divisor 24) in the default configuration—are then combined into a stereo downmix through Praat's multi-object stereo combination, which performs implicit summation and panning. The final output undergoes peak normalization to a user-specified level (default 0.99), preventing clipping while maximizing dynamic range. This architecture produces rich, metallic timbres and rhythmic artifacts, with the channel count and divisor relationships defining the spectral coloration and temporal texture of the processed sound.

Quick start

  1. In Praat, select a Sound object.
  2. Run script… → 8-Channel Delay.praat.
  3. Set parameters and click OK.
  4. Output object appears (see Outputs).
