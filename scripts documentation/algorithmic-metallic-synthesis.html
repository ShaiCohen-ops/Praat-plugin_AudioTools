What this does

Praat script.

Process Description: This script moves beyond simple FM by using gating (or "triggering") to simulate the hammer strike of a metallic object (like a bell, cymbal, or chime) followed by a damped exponential decay to model the resonance.

Technical details: This Praat script implements a gated-resonator synthesis engine that generates metallic and bell-like timbres through frequency-modulated carrier oscillators with pulsed excitation and exponential decay envelopes, mimicking the physical behavior of struck metal objects. The algorithm constructs multi-voice, multi-layer architectures where each voice represents a resonant mode with distinct frequency, decay characteristics, and temporal activation patterns. The synthesis pipeline operates in four stages per voice: first, a carrier oscillator is generated with frequency modulation (sin(2π × freq × detune × t + sin(2π × mod_rate × t) × mod_depth)) where the detune parameter creates slight frequency spreading across voices (typically 1-2.5% per voice) to prevent phase-locked summation and enhance spectral richness; second, a binary trigger signal is created using modulo arithmetic (if (t × rate) mod period < threshold then 1 else 0) to generate periodic rectangular gates that determine when each resonator is excited; third, the carrier is multiplied by the trigger to create gated bursts, then bandpass filtered at the carrier frequency with mode-specific bandwidths (150-400 Hz) to isolate spectral energy around the resonance and simulate Q-factor characteristics of physical resonators; fourth, an exponential decay envelope ((offset + scale × exp(-t/decay_time))) is applied to create the characteristic attack-less, exponentially decaying amplitude profile of struck idiophones. Five synthesis modes implement distinct voice count, frequency distribution, and temporal density strategies: Standard Metallic uses integer multiples of base frequency offset by (i+2) with 20% duty cycle triggers repeating at voice-dependent rates, creating a dense metallic texture with clear harmonic structure; Dense Shimmer doubles voice count with closer frequency spacing (1.5% detune), faster modulation (2× rate), shorter decay (0.6× time), and higher-frequency starting points, producing shimmering, rapidly decaying cascades; Sparse Bells halves voice count with extended decay (2× time), slower modulation (0.3× rate), and sparse triggers (30% duty cycle over 1.5-second periods), generating isolated bell-like strikes; Rhythmic Clang uses moderate voice count with frequent triggers (10% duty cycle over 0.5-second periods) and harmonic frequency relationships (i × base_freq) for percussive, rhythmic metallic patterns; and Chaotic Resonance randomizes all parameters (frequencies within 0.7-1.5× base with 0-3 partial offsets, modulation depths 2-5, decay times 0.5-1.5× specified, trigger periods 0.3-1.2 seconds) for unpredictable, non-repeating metallic textures. The frequency modulation depth (typically 2-5 radians) creates significant instantaneous frequency deviation, introducing spectral sidebands that enhance the inharmonic character essential to metallic timbres. Voice amplitudes use inverse square-root scaling (1/√N) to compensate for summation while allowing some constructive interference, with mode-specific base amplitudes (0.3-0.6) determining overall output levels. Four spatial modes provide stereo imaging: Mono preserves single-channel output; Wide adds 0.5 Hz sinusoidal modulation to the right channel creating subtle movement; Rotating implements circular panning at 0.3 Hz via quadrature amplitude modulation; Binaural applies 20-sample (~0.45ms) interaural delay with complementary filtering (left: 100Hz-4kHz, right: 300Hz-8kHz) for pseudo-3D localization. The script performs unnormalized additive synthesis allowing voice summation to create peaks, applies linear fade envelopes at boundaries, and includes optional peak normalization to 0.9. This architecture produces timbres ranging from smooth sustained metallic drones through sparse bell-like strikes to dense, shimmering cascades, suitable for synthesis of metallophones, cymbals, gongs, bells, and abstract metallic sound design where inharmonic spectral content, exponential decay, and temporal gating create the perceptual qualities of struck metal resonances without requiring computationally expensive physical modeling or modal synthesis algorithms.

Quick start

  1. In Praat, select a Sound object.
  2. Run script… → Algorithmic Metallic Synthesis.praat.
  3. Set parameters and click OK.
  4. Output object appears (see Outputs).
