What this does

Praat script.

Process Description: This script applies a sophisticated, iterative audio transformation inspired by quantum mechanics, specifically modeling the concepts of superposition and wave-function collapse. In each iteration (or "state"), the sound signal is blended with a time-shifted, phase-rotated copy of itself. The mixing ratio is governed by a decaying "superposition strength," which reduces the influence of the complex, phase-shifted components over time, simulating the effect of a state collapsing into a simpler form.

Quick start

  1. In Praat, select a Sound object.
  2. Run script… → Quantum_State_Superposition.praat.
  3. Set parameters and click OK.
  4. Output object appears (see Outputs).
