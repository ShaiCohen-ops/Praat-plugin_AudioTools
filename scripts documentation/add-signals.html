What this does

Praat script.

Process Description: The Stereo Balance and Volume Mixer script takes multiple selected Sound objects and combines them into a single Stereo Mix. Its primary function is to allow the user to control the Volume and Panning (left/right balance) of each sound file independently within the final mix, using specific gain multipliers for the left and right channels.

Technical details: This Praat script implements a multi-source stereo mixer with independent per-channel gain control, enabling precise balance and volume adjustment across up to five stereo sound objects. The algorithm operates through a channel-specific gain multiplication architecture: each input sound is first converted to stereo format if necessary (mono sources are duplicated to both channels), then processed through independent left and right channel gain stages where amplitude scaling is applied via the formula output_channel[n] = input_channel[n] × gain_coefficient. The gain coefficients (default 1.0 for unity gain, 0.0 for complete muting) are specified independently for each channel of each sound, providing 10 total gain parameters (ch1_1 through ch2_5) that control the stereo field distribution. The mixing process employs additive synthesis: a zero-initialized stereo buffer is created matching the duration and sample rate of the first selected sound, then each processed input is accumulated through sample-by-sample addition using Praat's formula operator self + object[tempSound], which performs element-wise summation across both channels simultaneously. This additive mixing preserves phase relationships and allows constructive/destructive interference between sources, with the potential for clipping if summed amplitudes exceed ±1.0 (no automatic gain compensation is applied). The channel-specific gain control enables sophisticated stereo positioning: setting left gain high and right gain low places a source toward the left speaker, equal gains maintain center positioning, and differential gain ratios create custom pan positions across the stereo field. The script processes sounds sequentially, applying gains through Praat's Formula (part) operator which targets specific channel ranges (channel 1 for left, channel 2 for right) without affecting the complementary channel, ensuring independent left/right manipulation. This architecture supports applications including stereo balance correction, multitrack mixing with pan control, creating custom stereo fields from mono sources through asymmetric gain distribution, and pseudo-spatial effects through time-varying gain relationships across multiple simultaneous sound sources.

Quick start

  1. In Praat, select a Sound object.
  2. Run script… → Add signals.praat.
  3. Set parameters and click OK.
  4. Output object appears (see Outputs).
