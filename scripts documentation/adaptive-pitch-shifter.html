What this does

Praat script.

Process Description: This script applies a single-step time-domain formula that results in a complex effect combining Frequency Modulation (FM) and Amplitude Modulation (AM). The instantaneous amplitude of the source sound dictates both the frequency and the volume of a newly synthesized sine wave, which is then blended with the original signal.

Technical details: This Praat script implements an amplitude-dependent ring modulation effect that creates dynamic spectral transformation through self-modulating synthesis rather than traditional pitch shifting. The algorithm applies the formula output[n] = input[n] × sin(2π × 440 × t × (base_shift + modulation_amount × |input[n]|)), where the modulating sinusoid's instantaneous frequency is controlled by the absolute value of the input signal itself, creating a feedback-like relationship between amplitude and spectral content. The base frequency of 440 Hz (A4) serves as the carrier, with the base_pitch_shift parameter (default 1.0) acting as a frequency multiplier and the modulation_amount parameter (default 0.5) scaling the amplitude-dependent frequency deviation. This produces a nonlinear, amplitude-to-frequency modulation (AFM) effect where louder passages increase the modulator frequency, creating denser zero-crossings and more complex sidebands at f_input ± (440 × (base_shift + mod_amount × |amplitude|)). The absolute value operation ensures unipolar modulation depth, preventing negative frequencies while allowing the modulator to track the input's envelope contour. This technique differs fundamentally from pitch shifting (which preserves harmonic relationships while transposing frequency) and instead generates ring modulation artifacts with time-varying spectral complexity: quiet passages produce relatively stable modulation at the base carrier frequency, while loud transients create rapid frequency sweeps and increased sideband density. The self-referential nature of the modulation—where the signal modulates its own spectral transformation—creates chaotic, metallic timbres with unpredictable harmonic content, particularly effective on percussive and dynamically varying source material. The result is characteristic of adaptive distortion or dynamic waveshaping, where the signal's amplitude envelope directly sculpts its spectral evolution through instantaneous frequency modulation of the ring modulator carrier.

Quick start

  1. In Praat, select a Sound object.
  2. Run script… → Adaptive Pitch Shifter.praat.
  3. Set parameters and click OK.
  4. Output object appears (see Outputs).
