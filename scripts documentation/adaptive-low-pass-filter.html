What this does

Praat script.

Process Description: This script applies a dynamic sine modulation filter to the selected sound. While the name suggests a traditional low-pass filter, the script executes a complex Amplitude Modulation (AM) process where the modulating frequency sweeps linearly over the duration of the sound. This creates a time-varying spectral effect, generating complex sidebands that result in a sweeping, metallic, or timbral effect.

Technical details: This Praat script implements a time-varying amplitude modulation effect that creates a pseudo-filtering sweep through sinusoidal multiplication rather than true frequency-domain filtering. The algorithm applies the formula output[n] = input[n] × sin(2π × f(t) × t), where f(t) = start_frequency + (end_frequency - start_frequency) × (t / duration) defines a linearly interpolated cutoff parameter that increases from the start frequency (default 200 Hz) to the end frequency (default 2000 Hz) over the signal's duration. However, despite the nomenclature, this operation does not implement a conventional low-pass filter with frequency-selective attenuation; instead, it performs ring modulation with a chirp (frequency-swept sinusoid) as the carrier, creating spectral sidebands at f_input ± f(t) for each frequency component in the source signal. The time-dependent modulation frequency causes the modulator phase to accumulate quadratically (∫f(t)dt = start_freq × t + ((end_freq - start_freq) / (2 × duration)) × t²), producing a continuously accelerating oscillation that generates characteristic swept-flange and tremolo artifacts. This creates amplitude nulls when the modulating sinusoid crosses zero, with the null rate increasing from the start frequency to the end frequency, effectively imposing a time-varying amplitude envelope with periodicities that sweep upward in frequency. The result is a dramatic timbral transformation characterized by increasing rhythmic pulsation, spectral sideband generation, and amplitude fluctuation rather than traditional low-pass filtering behavior. Following modulation, the script applies peak normalization to 0.99, rescaling the output to prevent clipping while preserving the relative amplitude contour shaped by the sinusoidal modulator. This technique is more accurately described as swept ring modulation or chirp-based amplitude modulation, useful for creating evolving tremolo effects, metallic coloration, and dramatic textural sweeps in sound design applications.

Quick start

  1. In Praat, select a Sound object.
  2. Run script… → Adaptive Low-pass Filter.praat.
  3. Set parameters and click OK.
  4. Output object appears (see Outputs).
