What this does

Praat script.

Process Description: This Praat script implements an Advanced Chaotic Modulation System for synthetic sound generation. Unlike traditional synthesis that uses simple, predictable waveforms (like single sine waves or fixed LFOs), this script aims to produce complex, non−periodic, and textures by simulating the behavior of mathematical Chaotic Attractors (like the Logistic Map, Lorenz, and Hénon) using combinations of highly detailed Frequency and Amplitude Modulation.

Technical details: This Praat script implements a multi-mode nonlinear synthesis engine inspired by chaotic dynamical systems, generating complex evolving timbres through nested sinusoidal modulation architectures that approximate the spectral characteristics of deterministic chaos. The algorithm constructs layered oscillator banks where each voice applies compound frequency and amplitude modulation using multiple sinusoidal modulators at related but non-integer frequency ratios, creating quasi-periodic trajectories that exhibit sensitivity to initial conditions characteristic of chaotic behavior. Five synthesis modes implement distinct modulation topologies: Chaotic Modulation combines optional frequency modulation at 3.7× the modulation rate (simulating logistic map periodicity), amplitude modulation with dual-rate components (0.5× and 1.3× rates mimicking Lorenz attractor projections), and secondary filtering modulation at 2.1× rate (approximating Hénon map dynamics); Logistic Frequencies applies coupled frequency modulators (sin(2π×3.7×mod_rate×t) and sin(2π×4.1×mod_rate×t)) with chaos-strength-weighted coefficients (1.5 and 0.8 respectively) to create frequency trajectories that traverse quasi-stable orbits; Lorenz Amplitudes uses three-component amplitude envelopes (0.4×sin(0.3×rate) + 0.4×sin(1.7×rate) + 0.2) with non-commensurate ratios to generate aperiodic amplitude fluctuations; Hénon Filtering applies single-modulator amplitude gating that creates rhythmic suppression patterns; and Combined Chaos implements full nested modulation with simultaneous frequency modulation, three-component amplitude modulation, and secondary amplitude gating for maximum spectral complexity. The frequency modulation depth is controlled by the chaos_intensity parameter, which scales the deviation magnitude, while modulation_rate determines the temporal density of parameter changes. Optional parameter randomization applies per-layer variation to base frequencies (60-140% of nominal) and chaos strength (70-130% of specified), ensuring each layer occupies distinct spectral and temporal regions to prevent masking. The synthesis employs direct formula-based generation at 44.1 kHz sample rate, with each layer's complete waveform equation compiled as a single Praat formula string and evaluated sample-by-sample—this approach avoids discrete control-rate quantization but limits system complexity due to formula length constraints. Four spatial modes (identical to the Brownian system) provide stereo imaging through complementary filtering, circular panning, or binaural simulation via asymmetric filtering and interaural delay. The script applies normalized layer summation without division (allowing constructive interference), linear fade envelopes at boundaries, and optional peak normalization to 0.9. Unlike true chaotic systems that require iterative state updates, this implementation uses predetermined modulation functions that approximate chaotic spectra through careful selection of non-rational frequency relationships and nested modulation depth—producing rich, evolving timbres with apparent unpredictability suitable for drone composition, experimental sound design, and generative music where controlled complexity creates perpetually interesting yet coherent sonic material without requiring computationally expensive chaos simulation algorithms.

Quick start

  1. In Praat, select a Sound object.
  2. Run script… → Advanced Chaotic Modulation.praat.
  3. Set parameters and click OK.
  4. Output object appears (see Outputs).
