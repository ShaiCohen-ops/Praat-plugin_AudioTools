What this does

Praat script.

Process Description: The Multi-channel Canon Generator script takes a single source Sound and generates three separate Stereo Canon pairs, resulting in a six-channel effect. This effect is achieved by creating copies of the original sound and applying distinct Pitch Shifts to each copy through a process of rate manipulation and resampling.

Technical details: This Praat script implements a multi-channel canon generator that creates layered audio variations through granular pitch shifting. The algorithm operates by first converting the input sound to mono and resampling it to a base frequency of 44.1 kHz. It then generates four independent pitch-shifted versions by manipulating the sampling frequency: the script calculates target sample rates by applying percentage-based pitch shifts (+12%, +6%, +19%, and -5.5% by default) to the base frequency, effectively creating pitch variations without time-stretching artifacts. Each shifted version is produced through a two-step process: first overriding the sampling frequency of the source audio (which alters pitch without changing duration), then resampling back to 44.1 kHz with a precision factor of 50 to maintain temporal alignment. The script employs a sophisticated channel architecture, generating three stereo pairs from the four shifted versions plus two additional copies of the original mono source. This results in six simultaneous audio streams (three stereo pairs) that create a complex canon texture. The stereo pairing strategy—(original+shift1), (shift2+shift3), and (shift4+original)—ensures balanced spatial distribution of the pitch variations. The script includes memory management through selective cleanup of intermediate processing objects, retaining only the primary output pair (canon_pair1) while optionally preserving all intermediate files for analysis. The resampling precision factor of 50 provides high-quality interpolation during frequency conversion, minimizing aliasing and spectral artifacts in the final output.

Quick start

  1. In Praat, select a Sound object.
  2. Run script… → 4-Channel Canon.praat.
  3. Set parameters and click OK.
  4. Output object appears (see Outputs).
