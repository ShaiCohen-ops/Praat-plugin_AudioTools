What this does

Praat script.

Process Description: This script executes a highly aggressive and technical spectral manipulation that achieves a "Spectral Knots" effect. The process involves converting the sound to the frequency domain (Spectrum), applying a non-linear formula that folds higher frequencies back into the spectrum based on a user-defined period, and simultaneously applying a complex, frequency-dependent amplitude modulation. This results in a synthetic, inharmonic, and metallic texture while selectively preserving the low-frequency energy of the original sound.

Quick start

  1. In Praat, select a Sound object.
  2. Run script… → Non-Linear Frequency Folding.praat.
  3. Set parameters and click OK.
  4. Output object appears (see Outputs).
